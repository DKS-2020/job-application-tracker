## 📌 Overview

Job Application Tracker is a full-stack productivity tool built for serious job hunters. Stop losing track of where you applied, what stage you're in, and which companies got back to you. Manage everything from one clean dashboard.

---

## ✨ Features

- 📝 **Add Applications** — Log every job you apply to with company, role, and date
- 📊 **Track Status** — Monitor each application through Applied → Interview → Selected / Rejected
- 🗂️ **Organized Dashboard** — See all applications at a glance
- 🔍 **Filter & Search** — Find specific applications quickly
- 🗑️ **Delete & Update** — Keep your tracker clean and up to date
- 💾 **Persistent Storage** — Data saved via backend so nothing is lost

## 🚀 Getting Started

### Prerequisites
- Node.js (v16 or above)
- npm

### Installation

```bash
# Clone the repository
git clone https://github.com/DKS-2020/job-application-tracker.git

# Navigate to project directory
cd job-application-tracker

# Setup Backend
cd backend
npm install
npm start

# Setup Frontend (new terminal)
cd frontend
npm install
npm start
```
